pm5-detail

Display all data provided by the Concept2 Performance Monitor 5 (PM5) using Bluetooth.

Requires

  • Computer (no mobile).
  • Bluetooth enabled.
  • Chrome web browser
  • PM5 only.

Usage

  • Click Connect above, then enable Bluetooth on the PM5.
  • Accept the connection on your computer.
  • Be patient; it takes a few seconds to connect.
  • Start row / skiing / biking!
  • Click boxes to highlight.
  • Drag boxes to rearrange.
